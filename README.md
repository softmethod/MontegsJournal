Программа обращается к журналу USN (UsnJrnl), извлекает время его создания и сопоставляет его со временем последней загрузки системы. На основе этого сравнения выводится результат проверки:

«Удалён» - если журнал был создан после запуска системы

«Не обнаружено» - если признаков удаления нет

«Ошибка» - если доступ к журналу получить не удалось
